# Baika Archery System Ver4.0

## 主な新機能

### 写真入力

- 的写真の読み込み
- ピンチズーム
- 1本指パン
- 2本指パン
- 写真回転
- 100%表示へのリセット
- 番号付きピンの追加
- ピンのドラッグ修正
- タップ式得点入力
- ①〜⑥の得点一覧
- 合計・X数・10以上のリアルタイム集計

### 表示同期

- 写真ピンを入力用の的へリアルタイム表示
- 写真ピンをグルーピングへリアルタイム表示
- ドラッグ中の連続同期
- 最後のピンを戻す操作を3画面で同期
- 写真を閉じた際の一括リセット

### クラウド保存

- 写真入力した6本をGoogleスプレッドシートへ保存
- 未ログイン時の部員名入力に対応
- 日付をJSTの yyyy-MM-dd 形式へ統一
- UTC変換による前日ずれを修正

### レスポンシブ表示

- PCの3列表示
- スマートフォン縦向きの1列表示
- スマートフォン横向きで写真と入力用の的を並列表示
- PC版グルーピングの固定表示
